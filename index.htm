Singularity.sg is a work in progress. If you are looking for the free jupyter notebook for deep learning please send a message on Discord server and provide your github profile. Once approved you can access the notebook at https://notebook.singularity.sg

Jupyter Notebooks you can choose from:

  • applejack: This is the recommended server. Minimal Scipy, pandas, good for everyone. Jupyter/scipy-notebook
  • fluttershy: applejack + tensorflow + keras Jupyter/Tensorflow-notebook
  • rainbowdash: fluttershy but GPU based (work in progress). This is purely experimental and the volume seems to differ from other Jupyter stacks. Which means your data is not copied to this server.

Data persistence. Please use the /work folder to save your files and data. Nothing outside that folder is retained across reboots. When you change the spawned container/server you data remains the same! To change container, hit "Control Panel", "Stop Server" and then "Start My Server". You can then select one of the docker images available.
